iPad work

The Problem

  • We want to be fully iPad compatible by BOY
  • Currently dragging is the only way to add a student to a NWT plan.
  • In iPads drag action pans the screen. Therefore, no way to add a student to a NWT plan.

Goals

  • Add to a NWT plan without dragging.
  • Because the iPad shrinks the screen by about 60% find a solution wherein users can zoom in greatly.
  • We'd prefer not to add many new elements or change layout of Matrix and LM in version 1.
