Guidelines

  1. Choose any of the options below to create your Peformance Task.
  2. Each Part is its own assessment. Together these Parts make up a single Performance Task.
  3. Each Part automatically comes with a Core Task. All activities in the Part relate to that Core Task.
  4. The Parts build upon each other. It is highly recommended that you adminster the Parts and activities in the order presented.
  5. It takes a 35-50 minute class period to complete a Core Task and just one activity. For that reason, it is recommended that you choose only one activity per Part.
